# Can I request a refund then?

You’re free to uninstall or downgrade at any time. Because MIDA keeps your paid access and data until the end of your current billing cycle — and reinstalling within that window restores everything — you usually keep the value you’ve already paid for without needing a refund. See [Downgrade, Cancel & Uninstall](https://docs.mida-app.io/recordings/subscription-and-billing/downgrade-cancel-uninstall/) for exactly what’s retained.

If you’ve run into a specific issue or would like to discuss a refund, please reach out to us directly via **Crisp Chat** or email us at [support@mida-app.io](mailto:support@mida-app.io?subject=%5BMIDA%20Support%5D%20Question%20about%20a%20Refund&body=Hi%20MIDA%20Team%2C%0A%0AI%20have%20a%20question%20about%20a%20refund.%20Please%20assist%20me%20with%20the%20following%3A%0A%0A-%20Shop%20URL%3A%20%0A-%20Issue%20details%3A%20%0A%0AThank%20you!%0A%0A%2D%20Your%20Name) — we’d like the chance to help and will work with you on the best solution.
