# Rule Priority & Execution Logic (Whitelist vs Blacklist vs VPN)

The priority order of the rules is Whitelist > Blacklist > Redirect.

![](https://docs.mida-app.io/_astro/unknown%20(15).Dn4fQrrJ_Z2hm7UR.webp)

In addition, you can manually arrange the execution order of rules in the **Settings** page >

**enable Manual Rule Ordering** \> drag and drop to set which rules have higher priority.

![](https://docs.mida-app.io/_astro/unknown%20(16).DjpTkNo6_Z1zY91t.webp)
