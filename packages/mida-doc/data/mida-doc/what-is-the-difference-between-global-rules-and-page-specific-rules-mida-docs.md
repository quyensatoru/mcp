# What is the difference between global rules and page-specific rules?

Global rules apply to all pages of your store. Once created, they affect every visitor regardless of which page they visit.

![](https://docs.mida-app.io/_astro/unknown%20(8).BYJYfeqW_1tgXl5.webp)

Page-specific rules, on the other hand, only apply to selected pages. You can use them when you want different settings or behaviors for specific pages (e.g., product pages, collection pages, custom paths).

![](https://docs.mida-app.io/_astro/unknown%20(11).DHp-MnMx_ZPzoWl.webp)

![](https://docs.mida-app.io/_astro/unknown%20(13).D8c9TSjV_ZOx8KM.webp)
