# Create Surveys

### **To create a survey in MIDA, please follow these steps:**

[Section titled “To create a survey in MIDA, please follow these steps:”](https://docs.mida-app.io/recordings/surveys/create-surveys/#to-create-a-survey-in-mida-please-follow-these-steps)

**Step 1:** Go to the “ **Survey**” section of the app.

![](https://docs.mida-app.io/_astro/image%20(38).D-k9BheB_ZtyDy4.webp)

**Step 2:** Click on the “ **Create survey**” button.

![](https://docs.mida-app.io/_astro/image%20(39).BkCJHd_-_Z1OEKhh.webp)

**Step 3:** Choose type of Survey

![](https://docs.mida-app.io/_astro/image%20(40).Cx3A_7sr_OSPUw.webp)

- Multiple choice: You can set up one question with different choices.
- Like or Dislike: You can set up one simple question with only Like or Dislike response.

Then click on the button **Next** to set up the Survey.

**Step 4:** Customize the **General Settings**

4.1. Fill the **General Information.**

![](https://docs.mida-app.io/_astro/image%20(41).Cgw0g3m7_1BtiLB.webp)

4.2. Choose **pages** to display survey (You can choose as many pages as you want)

![](https://docs.mida-app.io/_astro/image%20(42).T3zsRZ2N_Z2rWzvK.webp)

4.3. Customize the **survey settings** such as display devices, delay time, and position.

![](https://docs.mida-app.io/_astro/image%20(43).BoCreMSo_Z103cEk.webp)

**Step 5:** Customize the **Display Settings**

5.1. For Multiple choice Survey:

- Customize the **Question page**:

![](https://docs.mida-app.io/_astro/image%20(44).BBSAyE5N_ZJFIiK.webp)

You can have maximum 5 choices and minimum 2 choices. Once you **Allow multiple answers to be selected**, your customers can select multiple choices from choice list.

![](https://docs.mida-app.io/_astro/image%20(45).rU1y9MYV_1lWVA7.webp)

- Customize the **Thank you page**:

![](https://docs.mida-app.io/_astro/image%20(46).zwFSKcL2_Z10wFFS.webp)

5.2. For Like or Dislike Survey:

- Customize the **Question page**:

![](https://docs.mida-app.io/_astro/image%20(47).CvFsvLLX_Z1B1DNH.webp)

- Customize the **Thank you page**:

You can set up another message when customers feedback as Dislike by enable the toggle button **Show a different message when feedback is Dislike.**

![](https://docs.mida-app.io/_astro/image%20(48).BjCGTEoX_ZIvjUH.webp)

**Step 6: Save** the survey and activate it on your website.

![](https://docs.mida-app.io/_astro/image%20(49).D6H5oC2P_Z2a7BDg.webp)

Once your survey is activated, you can monitor the results and analyze the data to gain insights into your customers’ preferences and needs.
