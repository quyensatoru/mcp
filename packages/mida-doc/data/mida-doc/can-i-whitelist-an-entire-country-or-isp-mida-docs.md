# Can I whitelist an entire country or ISP?

You can whitelist an entire country or ISP to allow all visitors from that region or network.

When creating a rule, select **Whitelist** as the rule type, then choose **Country** or **ISP** as the condition. Specify the country or ISP you want to allow, and all visitors matching that condition will be whitelisted automatically.
