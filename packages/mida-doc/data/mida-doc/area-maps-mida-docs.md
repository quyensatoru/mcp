# Area maps

Area Heatmap displays the total clicks for all the elements within a selected area. Unlike a click map, you can choose the area and view the heat for all the elements within the area.

![](https://docs.mida-app.io/_astro/area%20maps.fWs0x_r5_2rQdXG.webp)

## Top panel

[Section titled “Top panel”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#top-panel)

The top panel helps you filter the Click maps data and actions for them.

1. **URL:** This autofill URL based on the page you choose from the Heatmap list page

![](https://docs.mida-app.io/_astro/image%20(7).CX7cvyBV_ZtO4XU.webp)

2. **Device:**

You can choose what device to see heatmap, including PC, Tablet, and Mobile. The number on the right side of it is the number of pageviews of that URL on that device.

![](https://docs.mida-app.io/_astro/image%20(5).B0Wth3lZ_ZY2O7n.webp)

3. **Heatmap type:** including 4 types

- Clicks
- Moves
- Scrolls
- Areas

4. **View recordings:**

View recordings help you watch all sessions that included this URL.

![](https://docs.mida-app.io/_astro/view%20recordings%20click%20maps.Ce4VIhb-_ZsP8Bh.webp)

## Heatmap panel

[Section titled “Heatmap panel”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#heatmap-panel)

![](https://docs.mida-app.io/_astro/image%20(25).Bw93loU9_Z1urpGd.webp)

### View Area map

[Section titled “View Area map”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#view-area-map)

View default areas based on the heat activity with applied filters. Distinct areas have a defining boundary, and CTR is visible by default on the area itself.

**CTR**: Click Through Rate (CTR) is the number of clicks on the area present by the total number of clicks on matched pages.

![](https://docs.mida-app.io/_astro/image%20(26).ClO3SmeF_2sUAzO.webp)

**Note**:

- The area could be on the whole page based on what you selected.
- Areas could also have no click activity.

### Edit Area map

[Section titled “Edit Area map”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#edit-area-map)

In this mode, you have the ability to select new regions on the area maps. Select the ‘Edit heatmap’ button in the sidebar panel to switch from viewing to editing area maps.

![](https://docs.mida-app.io/_astro/image%20(27).BiUQ35_S_ZdR8tQ.webp)

You can select an area by clicking on the heatmap. The number of clicks and CTR for the new area is calculated on the tooltip. You can also remove an existing area.

![](https://docs.mida-app.io/_astro/edit%20areas.Ld4UnD7W_2mxg9e.webp)

Select ‘Exit edit’ when you finish editing. Then the new Area map will be shown.

At the bottom of Heatmap panel, there are:

1. **Clicks:** the total number of clicks on a page for the applied device.
2. **Color scale**: Colors get warmer as the popularity of a section increases. The warmer the colors, the denser the clicks.

![](https://docs.mida-app.io/_astro/image%20(3).BsWRCZrm_XcEp8.webp)

## Sidebar panel

[Section titled “Sidebar panel”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#sidebar-panel)

The sidebar panel contains the list of areas in order of the number of clicks in the default area maps. Selecting a new area in Edit mode will add it to the top of sidebar.

### Area click data

[Section titled “Area click data”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#area-click-data)

As you scroll to the bottom, you see elements with the number of clicks (% of clicks). Once you select an element on the Sidebar panel, it will automatically navigate you to that element on the Heatmap panel.

![](https://docs.mida-app.io/_astro/sidebar%20area.DhbuWmXc_ZdDrmv.webp)

### View recordings

[Section titled “View recordings”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#view-recordings)

View recordings help you watch recordings where the element was clicked.

![](https://docs.mida-app.io/_astro/image%20(28).CPpq-ml9_Z1Pfm2f.webp)

### Copy element

[Section titled “Copy element”](https://docs.mida-app.io/recordings/heatmaps/area-maps/#copy-element)

This copies the CSS selector of an element that allows you to identify the exact element in the UI. You can use it to change the style of an element.

![](https://docs.mida-app.io/_astro/image%20(29).G6k1QXsi_Zwwytz.webp)
