# Can I change the number of recordings per page?

In the Session replays, you can change the number of recordings per page by selecting it in the drop up list at the end of page.

![](https://docs.mida-app.io/_astro/image%20(340).DUF3_zJ0_Z26Pfba.webp)

You can see up to 50 recordings per page. It can be useful when you do a mass action.

Please be careful when mass delete recordings because once you delete, they cannot be restored.
