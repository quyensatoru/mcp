# Enable app embed

You need to enable app embed block for in-app features can work properly.

![](https://docs.mida-app.io/_astro/appembed1.DFTJ7xKB_2iD8Ne.webp)

**Step 1:** Click on the button “ **Enable app”** in the banner.

You will be redirected to the page of current theme editor with MIDA app embed is enabled.

**Step 2:** Make sure that MIDA app embed is enabled and click “ **Save”**.

![](https://docs.mida-app.io/_astro/appembed2.DI-0wcN7_Z1Sm5kK.webp)

Please note that if you change your theme, you need to enable MIDA app embed again.
