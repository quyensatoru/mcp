# App Embedded

First steps after installing the MIDA app:

1. **Enable Theme app extension:**

Enable theme app extension to block access store

![](https://docs.mida-app.io/_astro/unknown%20(19).CL8_s2hv_ZdrPXB.webp)

Click **Save**.

![](https://docs.mida-app.io/_astro/unknown%20(20).2Q9r91Hc_Z1fNdAR.webp)

After completing **Step 1**, it will be marked as **Done**.

![](https://docs.mida-app.io/_astro/unknown%20(21).IljpGY7e_Z2lGXQ6.webp)

2. **Create your rule:**

![](https://docs.mida-app.io/_astro/unknown%20(22).CMnpBZz4_VCFTf.webp)

At this step, you need to create your first rule: Go to Rules → click New Rule → set up the rule → then click Save.

After completing **Step 2**, it will be marked as **Done**.

![](https://docs.mida-app.io/_astro/unknown%20(23).DUtj-XIU_Z1ndasA.webp)

3. **Review settings**

Refer to the VPN/Proxy/TOR blocking features in the Settings tab.

![](https://docs.mida-app.io/_astro/unknown%20(24).CA10imle_2jue1e.webp)

4. **Review visitor analytics**

Review visitor analytics, then create the appropriate rule.

![](https://docs.mida-app.io/_astro/unknown%20(25).DOqmZwJF_1KwdR2.webp)

5. **Enable protected content**

Prevent customers from bypassing app blocking logic

![](https://docs.mida-app.io/_astro/unknown%20(26).JXGML47f_1oLCgX.webp)

6. **Update template block page**

Customize content template according to your preferences

![](https://docs.mida-app.io/_astro/unknown%20(27).BB9m2eLP_Z1UYOSP.webp)

When all steps are completed, the status will display “ **Onboarding Completed**”, and the widget **will no longer be shown.**

![](https://docs.mida-app.io/_astro/unknown%20(28).BIX_vhFs_Z15XOTR.webp)
