# How to Customize Block Page

How to Customize the Block Page:

Go to **Settings** \> **Block Page Settings** \> then configure the settings for your Block Page.

![](https://docs.mida-app.io/_astro/unknown%20(17).BEjAMQkp_27Ob7y.webp)

- Block Page Title: Set the title displayed on the block screen
- Block Page Message: Set the detailed message explaining the reason for being blocked
- Logo/Image: Set the brand logo on the block page
- Background Color: Set the background color of the block page
- Enable Overlay Mode: Show an overlay on the block page.
