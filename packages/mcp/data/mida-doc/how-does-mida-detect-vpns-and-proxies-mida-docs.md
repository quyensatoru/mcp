# How does MIDA detect VPNs and proxies?

MIDA detects VPNs and proxies by looking up IP information from [proxycheck.io](http://proxycheck.io/).

[proxycheck.io](http://proxycheck.io/) is an online IP lookup service that helps detect whether an IP address is using a Proxy, VPN, TOR, or has been compromised.
