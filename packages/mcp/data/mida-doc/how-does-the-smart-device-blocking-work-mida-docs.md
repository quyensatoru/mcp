# How does the "Smart Device Blocking" work?

The **Smart Device Blocking** feature works by detecting devices that change their IP address after a previous visit. If a device has visited before and later switches to a fake VPN, MIDA app will recognize the IP change and block access from that device.

The purpose of this feature is to help prevent fraudulent access.
