# Can I allow staff/admin even if their location is blocked?

Yes, you can.

![](https://docs.mida-app.io/_astro/image%20(1).BL3pLkZH_Z2o25WP.webp)

Simply use the Admin Access URL feature in MIDA. This special URL allows staff/admins to access the store even if their IP, location, or VPN is blocked.

Go to: Settings → Admin Access → Input & Copy your unique admin URL.

Use that link to bypass all block rules.
